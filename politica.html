Política de Privacidade

Última atualização: 02 de Dezembro de 2025

01 Coleta de Dados e Permissões

Para fornecer a funcionalidade de "Allowlist" (Lista de Permissões) e "Blocklist" (Lista de Bloqueio), o Focus Shark solicita as seguintes permissões específicas:

  • Identity (Autenticação): Utilizamos a autenticação do Google apenas para verificar a validade da sessão.
  • YouTube Data API (Escopo youtube.readonly): Acessamos estritamente a lista de canais nos quais você está inscrito.
    Dados coletados: Nome do canal e URL da imagem de perfil (avatar).

* O uso da API é necessário para garantir a estabilidade da interface visual, evitando erros de leitura (CSS scraping) quando o YouTube atualiza seu layout.

02 Armazenamento de Dados

Sua privacidade é nossa prioridade absoluta. A arquitetura do Focus Shark foi desenhada para ser Local-First.

Armazenamento Local

Todos os dados (avatares, nomes e preferências) são salvos apenas no seu navegador via chrome.storage.local.

Sem Servidores Externos

Nós não possuímos banco de dados. Nenhum dado seu sai da sua máquina para nossos servidores.

03 Compartilhamento e Conformidade

Nós não vendemos, trocamos ou transferimos suas informações pessoais para terceiros.

O uso de informações recebidas das APIs do Google pelo Focus Shark adere à Política de Dados do Usuário dos Serviços de API do Google, incluindo os requisitos de "Uso Limitado".
